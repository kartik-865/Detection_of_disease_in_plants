🌱 About This Project

Plant Disease Detection is a deep learning system that classifies plant leaf images into 38 categories spanning 14 crop species, using the PlantVillage dataset. The model is a custom Convolutional Neural Network trained end-to-end with data augmentation to achieve validation accuracy consistently above 90%.

🧠 Model Architecture

🖼️
Input Layer
Accepts RGB images at 224 × 224 resolution, normalised to [0, 1].
🔍
Feature Extraction
4× Conv2D blocks (32→64→64→128 filters) with BatchNorm, MaxPooling, and Dropout (0.2).
🎯
Classification Head
Dense(128, relu) → Dense(64, relu) → Dense(38, softmax). Trained with Adam + sparse categorical cross-entropy.
🔄
Data Augmentation
Random flipping, rotation, zoom, and brightness adjustments to improve generalisation.
